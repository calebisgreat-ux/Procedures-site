    Welcome 👋

    This is a clean-room template that mimics the general idea of a procedures knowledge base—without copying any proprietary content or design.

    Use the sidebar to pick a sample item, or add your own in data/procedures.json. The search bar filters the list instantly.

    Custom link: Set your desired URL in config.js under NEW_LINK. The button in the top-right will open it.

    Features

    • Instant search/filter
    • Client-side routing via hash (e.g. #/procedure/onboarding)
    • Responsive layout + light/dark theme
    • Keyboard: / focuses search, Esc clears
